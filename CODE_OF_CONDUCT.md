# Code de conduite

Baloo est un projet bénévole pour bénévoles. La règle est simple : **on est sympas, on s'entraide, on respecte les autres**.

## Engagements

Pour faire de cet espace un lieu accueillant, nous nous engageons à :

- accueillir les contributions de tous niveaux, sans condescendance ;
- respecter les points de vue et expériences différentes ;
- accepter les critiques constructives avec ouverture ;
- privilégier l'intérêt du projet et de ses utilisateurs (trésoriers bénévoles SGDF) ;
- faire preuve d'empathie envers les autres membres de la communauté.

## Comportements inacceptables

- Langage ou imagerie à caractère sexuel ou discriminatoire.
- Attaques personnelles, trolling, commentaires insultants.
- Harcèlement public ou privé.
- Publication d'informations privées (doxxing) sans consentement.
- Toute conduite qui serait raisonnablement considérée comme inappropriée dans un cadre professionnel.

## Application

Les mainteneurs sont responsables de clarifier les standards et prendront des mesures correctives appropriées et justes en cas de comportement inacceptable.

Ils peuvent retirer, modifier ou refuser les commentaires, commits, code, issues, et autres contributions non alignées avec ce code de conduite.

## Signalement

Les comportements abusifs, harcelants, ou autrement inacceptables peuvent être signalés en contactant l'auteur du projet via GitHub (`@benomite`). Toutes les plaintes seront examinées et donneront lieu à une réponse jugée nécessaire et appropriée aux circonstances. Confidentialité respectée.

## Référence

Ce code de conduite est inspiré du [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
